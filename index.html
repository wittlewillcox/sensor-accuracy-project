Sensor Accuracy Experiment

Comparing Pressure, Ultrasonic, and Mechanical Sensors

Purpose Variables Sensors Materials Results

Purpose

The purpose of this science fair project is to compare the accuracy of different sensor types under the same conditions. I wanted to do this as a project because it solves a problem I have, not knowing if my animals have water or not. And i knew that this would help other people as well.

Variables

Independent Variable

Type of sensor used

Dependent Variable

Accuracy of the sensor measurements

Controlled Variables

Distance, the board used, environment

Sensors Used

  • Pressure Sensor
  • Ultrasonic Sensor
  • Mechanical Sensor

Materials used

  • Arduino Nano
  • Sensor of one of the sorts
  • Jumper Wires
  • Breadboard
  • A PC with Arduino IDE
